# MAIIA TEST

Technical test React + Redux.

## Installation

Just need Git and Yarn or NPM to install.

```bash
git clone https://github.com/Devkidd/maiia-test 
```

## Install packages and run the app 
```bash
yarn install
```

```bash
yarn start
```

Or if you use NPM

```bash
npm install
```

```bash
npm start
